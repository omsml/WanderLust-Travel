## 🧭 About the Project

**WanderLust Travel** is a professional, vibrant, and fully responsive travel agency website designed as a college assignment project. It demonstrates real-world web development skills including layout design, responsiveness, interactivity, UI/UX best practices, and form validation — all without any external frameworks or libraries.

> Built to fulfill a 5-page travel agency website assignment with modern UI/UX design, clean file structure, and all suggested features implemented.

---

## 🛠 Tech Stack

| Technology | Usage |
|------------|-------|
| **HTML5** | Semantic page structure, forms, accessibility attributes |
| **CSS3** | Custom properties (variables), Grid, Flexbox, animations, media queries |
| **Vanilla JavaScript** | DOM manipulation, IntersectionObserver, form validation, slider, accordion |
| **Google Fonts** | `Playfair Display` (display/headings) + `DM Sans` (body text) |
| **Unsplash** | Free high-quality travel photography (no API key needed) |

## 🎨 Color Palette

| Name | Hex | Usage |
|------|-----|-------|
| 🔴 Coral | `#FF5A5F` | Primary CTA, badges, accents |
| 🔵 Ocean | `#0099CC` | Secondary buttons, links |
| 🟡 Sun | `#FFB347` | Highlights, hero text accents |
| 🟢 Mint | `#00C9A7` | Success, feature checkmarks |
| 🟣 Violet | `#7C5CBF` | Section tags, gradients |
| ⬛ Dark | `#0D1B2A` | Background dark sections, navbar |
| ⬜ Light | `#F8F6F0` | Page background |

### Gradients
```css
--grad1: linear-gradient(135deg, #FF5A5F 0%, #FFB347 100%);   /* Coral → Sun */
--grad2: linear-gradient(135deg, #0099CC 0%, #00C9A7 100%);   /* Ocean → Mint */
--grad3: linear-gradient(135deg, #7C5CBF 0%, #FF5A5F 100%);   /* Violet → Coral */
```
